# Requirements pip-installed into the AppImage's bundled Python, in order.
#
# photonfdtd (the FDTD solver) isn't on PyPI, so it's fetched from GitHub. It's
# listed first so it's installed before phidler[fdtd] resolves its
# `photonfdtd>=0.2.0` pin (which no PyPI release satisfies).
photonfdtd @ git+https://github.com/ngpaladi/photonfdtd.git
#
# phidler itself, with the [fdtd] extras (matplotlib + numba), is appended by
# packaging/appimage/build.sh as an absolute path to the current checkout — so
# the AppImage bundles this exact source, not a fixed git ref. The extras are
# bundled at build time on purpose: an AppImage is a read-only mount, so the
# app's "install photonfdtd on demand" pip path can't run inside it — everything
# FDTD needs has to be here already.
